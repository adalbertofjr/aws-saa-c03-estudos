# Estudos — AWS Certified Solutions Architect – Associate (SAA-C03)

Material de estudo pessoal para a certificação SAA-C03, em português. Cada aula do AWS Skill
Builder é transformada em um **dossiê de cinco artefatos**, e o conjunto é publicado como
site para estudar de qualquer lugar.

**📖 [Acessar o site](https://adalbertofjr.github.io/aws-saa-c03-estudos/)**

## Os cinco artefatos

| # | Arquivo | O que é |
|---|---|---|
| 1 | `01-resumo.md` | Uma página orientada a decisão — leitura de véspera |
| 2 | `02-conteudo.md` | A aula reescrita como material de leitura |
| 3 | `03-servicos.md` | Tabela de serviços com o discriminador de cada um |
| 4 | `04-mapa-mental.md` | Mapa mental em Mermaid |
| 5 | `05-flashcards.csv` | Flashcards — importáveis no Anki e lidos pelo site |

Mais dois acumuladores que crescem a cada aula: o [índice de
serviços](_estudo/00-indice-servicos.md), que vira a folha de consulta única da semana do
exame, e um mapa mental por domínio.

## Como funciona

O processo está descrito em [`CLAUDE.md`](CLAUDE.md) e segue **RPI** (Research → Plan →
Implement). Dois princípios que moldam o resto:

- **Procedência explícita.** Todo fato carrega sua origem: sem marca = veio da aula ·
  `†` = conhecimento não verificado · `[doc]` = confirmado na documentação AWS, com link ·
  `⚠️ LACUNA` = não foi possível confirmar. Números nunca são inventados.
- **O CSV é a única fonte dos flashcards.** O Anki importa o arquivo e o site lê o mesmo
  arquivo. Nada é duplicado.

## Flashcards

O site tem um player com repetição espaçada (SM-2 simplificado) e progresso salvo no
navegador. É para revisar em qualquer lugar — o agendamento sério continua no Anki, já que o
progresso do site não sincroniza entre dispositivos (há exportação manual em JSON).

Filtre por `disc` para treinar só discriminação entre serviços, que é o que o exame mais cobra.

## Sobre as transcrições

**As transcrições dos cursos não estão neste repositório e não estarão.** São material
licenciado de cursos pagos da AWS; redistribuí-las seria indevido. Elas ficam apenas na
máquina local (ver `.gitignore`). O que se publica aqui é o material derivado — resumos,
notas, tabelas e cards que escrevi a partir das aulas.
